# 서비스 규칙 (Service Rules)

기능 구현 시 위반해서는 안 되는 도메인 불변식.

---

## GPS / 위치 서비스

### 원칙

1. **GPS는 강제하지 않는다.** 앱 진입·화면 이동 시 GPS를 자동 측정하지 않는다.
2. **사용자가 명시적으로 "내 현재 위치"를 선택한 시점에만** GPS를 측정하고 해당 Ward를 자동 결정한다.
3. 수동으로 구/동을 탭해 선택하는 것도 위 정책과 동일하게 동작한다 — GPS 불필요.

### 기본 상태 (아무것도 선택 안 했을 때)

- **위치 미선택** = 전체 지역 노출. 필터 없음.
- 마켓: 전체 매물 표출 (`wardId=null, districtId=null`).
- 지도: 구별 카운트 배지만 표시, 특정 동네 선택 없음.

### 화면별 적용

| 화면 | 기본값 | 위치 선택 방법 |
|---|---|---|
| 마켓(MarketMain) | 전체 지역 | 헤더 지역명 탭 → 시트에서 선택 또는 현재 위치 |
| 지도(NeighborhoodMap) | 미선택 | 지도에서 동네 탭 또는 `◎` 버튼(현재 위치) |
| 물건 등록 | 위치 없음 | 명시적 위치 선택 시에만 좌표 저장 |
