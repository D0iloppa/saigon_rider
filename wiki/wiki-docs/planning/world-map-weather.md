# 날씨 & 대기질 정보 서비스

## 1. 서비스 개요

오토바이 라이더에게 **출발 전 날씨 판단**은 가장 중요한 일상 루틴이다.
비 예보 확인, 체감 온도, 바람 세기, 대기질까지 한 화면에서 제공한다.

| 정보 | 라이더 활용 시나리오 |
|------|---------------------|
| 현재 기온 + 체감 | 복장 결정 (재킷, 장갑 등) |
| 시간별 강수확률 | 우비 준비, 출발 시간 조정 |
| 바람 속도/방향 | 고속도로 주행 안전 판단 |
| 습도 | 헬멧 바이저 김 서림 대비 |
| AQI (대기질지수) | 마스크 착용 여부 결정 |

## 2. 위젯 구성

### 2-1. 현재 날씨 섹션

```
┌──────────────────────────────┐
│  🌤  Ho Chi Minh City          │
│                                │
│     32°              ☁️       │
│     체감 36°  흐림             │
│                                │
│  💨 3.2 m/s    💧 75%          │
├──────────────────────────────┤
│  AQI  87  보통                 │
│  ████████░░░░  (색상 바)        │
└──────────────────────────────┘
```

- **기온**: 큰 숫자 (48px+), 체감 온도는 부제로 표시
- **날씨 아이콘**: 맑음/흐림/비/뇌우 등 조건에 따라 변경
- **바람 & 습도**: 작은 칩 형태로 한 줄에 배치
- **비 예상 시**: "비 예상" 경고 칩 강조 표시 (빨간/주황 컬러)

### 2-2. 대기질(AQI) 섹션

| AQI 범위 | 등급 | 색상 | 라이더 행동 가이드 |
|----------|------|------|-------------------|
| 0-50 | 좋음 (Good) | 초록 | 마스크 불필요 |
| 51-100 | 보통 (Moderate) | 노랑 | 민감군 마스크 권장 |
| 101-150 | 민감군 나쁨 | 주황 | 마스크 착용 권장 |
| 151-200 | 나쁨 (Unhealthy) | 빨강 | 마스크 필수, 장시간 주행 자제 |
| 201+ | 매우 나쁨 | 보라 | 외출 자제 권고 |

- 색상 바로 시각적 즉시 판단 가능
- AQI 수치 + 등급 텍스트 함께 표시

### 2-3. 시간별 예보 (가로 스크롤)

```
 14시    15시    16시    17시    18시    19시  →
  ☀️      ⛅      🌧️     🌧️     ⛅      ☁️
 33°     32°     30°    29°     28°    27°
 10%     30%     80%    75%     40%    20%
```

- 현재 시각부터 12시간 표시
- 각 시간: 아이콘 + 기온 + 강수확률(%)
- 강수확률 60% 이상 시 해당 시간 슬롯 강조 (배경 색상 변경)
- 가로 스와이프로 탐색

## 4. API 후보 비교

| 서비스 | 무료 한도 | 시간별 예보 | AQI | 비용 (유료) |
|--------|----------|------------|-----|------------|
| **WeatherAPI.com** | 1M calls/월 | O (3일) | O | $9/월~ |
| OpenWeatherMap 2.5 | 60 calls/분 | X (3시간 간격) | 별도 API | $0 |
| OpenWeatherMap 3.0 | 1000 calls/일 | O | 별도 API | 종량제 |
| Visual Crossing | 1000 calls/일 | O | X | $35/월~ |

**권장**: WeatherAPI.com — 무료 한도가 넉넉하고 시간별 예보 + AQI를 하나의 API에서 제공

## 5. 캐싱 전략

```
유저 요청 → BFF 서버 → Redis 캐시 확인
                         ├── HIT → 캐시된 데이터 반환 (< 5ms)
                         └── MISS → 외부 API 호출 → Redis 저장 (TTL 10분) → 반환
```

- **캐시 키**: `weather:hcmc:{lang}` (언어별 분리 — 날씨 설명이 다국어)
- **TTL**: 10분 (날씨 데이터의 적절한 신선도)
- **일일 API 호출 예상**: 최대 144회/언어 × 3언어 = 432회 (무료 한도 대비 0.04%)

## 6. 에러/예외 처리

| 상황 | UI 대응 |
|------|---------|
| API 응답 대기 중 | 스켈레톤 shimmer 표시 |
| 외부 API 오류 | "날씨 정보를 불러올 수 없습니다" + 재시도 버튼 |
| 네트워크 끊김 | 동일한 에러 카드, 재시도 시 자동 복구 |
| 캐시된 데이터가 오래됨 (30분+) | 정상 표시하되 "30분 전 기준" 표시 |
