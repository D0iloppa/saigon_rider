# 유가 정보 서비스

## 1. 서비스 개요

베트남의 오토바이 연료비는 라이더의 **일상 최대 변동 비용**이다.
정부 고시 가격이 약 10일 간격으로 변경되며, 연료 종류별 가격 차이가 리터당 1,000~2,000 VND에 달한다.
이 정보를 앱에서 바로 확인할 수 있으면 주유 타이밍과 연료 선택에 도움이 된다.

## 2. 베트남 연료 체계

### 주요 연료 종류

| 연료 | 용도 | 특징 |
|------|------|------|
| **RON 95** | 고성능 차량/일부 대형 오토바이 | 가장 비쌈, 옥탄가 높음 |
| **RON 92** | 일반 오토바이 (가장 보편적) | 표준 선택지 |
| **E5 RON 92** | 바이오에탄올 5% 혼합 | 정부 권장, 가장 저렴 |

> 대부분의 오토바이(Honda Wave, Air Blade, Vision 등)는 RON 92 또는 E5 RON 92 사용

### 가격 결정 구조

- **정부 고시 가격**: 산업무역부(Bo Cong Thuong)가 결정
- **변경 주기**: 약 10일 (매월 1일, 11일, 21일 전후)
- **주요 브랜드**: Petrolimex, PV Oil, Saigon Petro
- **브랜드 간 가격 차이**: 동일 연료 기준 50~200 VND 이내

## 3. 위젯 구성

```
┌──────────────────────────────┐
│  ⛽  유가 정보                 │
│                                │
│  RON 95       25,420 ₫/L      │
│  RON 92       24,320 ₫/L      │
│  E5 RON 92    23,850 ₫/L      │
│                                │
│  2026-05-15 기준               │
└──────────────────────────────┘
```

- **가격 표시**: VND 단위, 천 단위 콤마 구분 (예: 25,420)
- **단위**: /L (리터당)
- **기준일**: 마지막 가격 갱신일 표시
- 가격 변동 시 **↑ / ↓ 화살표** + 변동폭 표시 (Phase 2)

## 4. 데이터 관리 방식

### Phase 1: 관리자 수동 입력

```
관리자 콘솔 → 유가 등록 폼 → DB 저장 → 프론트 API 조회
```

| 항목 | 설명 |
|------|------|
| 입력 주체 | 운영자 (관리자 콘솔) |
| 입력 주기 | 정부 고시 발표 시 (~10일 간격) |
| 데이터 소스 | petrolimex.com.vn, xangdau.net |
| 저장소 | PostgreSQL `fuel_prices` 테이블 |

### DB 스키마

| 컬럼 | 타입 | 설명 |
|------|------|------|
| id | SERIAL | PK |
| fuel_type | VARCHAR(30) | RON_95, RON_92, E5_RON_92 |
| price_vnd | INTEGER | 리터당 가격 (VND) |
| effective_date | DATE | 적용일 |
| source | VARCHAR(200) | 데이터 출처 |
| note | TEXT | 비고 |

### Phase 2: 자동화 (향후)

- 정부 고시 사이트 크롤링 자동화
- 주유소별 실시간 가격 (브랜드별 차이 표시)
- 주유소 위치 지도 (가까운 주유소 안내)

## 6. API 엔드포인트

### 공개 API (인증 불필요)

| 메서드 | 경로 | 응답 |
|--------|------|------|
| GET | `/api/world-info/fuel-prices` | 최신 연료별 가격 목록 |

**응답 예시**:
```json
{
  "prices": [
    { "fuel_type": "RON_95", "price_vnd": 25420, "effective_date": "2026-05-15" },
    { "fuel_type": "RON_92", "price_vnd": 24320, "effective_date": "2026-05-15" },
    { "fuel_type": "E5_RON_92", "price_vnd": 23850, "effective_date": "2026-05-15" }
  ],
  "last_updated": "2026-05-15"
}
```

### 관리자 API

| 메서드 | 경로 | 기능 |
|--------|------|------|
| GET | `/admin/fuel-prices` | 유가 관리 페이지 렌더링 |
| POST | `/admin/fuel-prices` | 신규 가격 등록 |
| POST | `/admin/fuel-prices/delete` | 가격 삭제 |
